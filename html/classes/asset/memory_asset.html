Memory Asset class¶

class starfish.asset.MemoryAsset(metadata=None, did=None, data=None)¶

Bases: starfish.asset.asset_base.AssetBase

Memory asset can be used to try out assets without accessing any file system

Parameters

  • data (str or byte array) – data string or byte text to save as the asset

  • metadata (None or dict) – Optional dictionary metadata to provide for the asset if non used then the class will generate a default metadata based on the data provided

  • did (None or str) – Optional did of the asset if it’s registered

property data¶

Return the asset data

Returns

the asset data

Type

str or byte

save(filename)¶

Saves the data in the memory asset to a file.

Parameters

filename (str) – Filename to save the data.
