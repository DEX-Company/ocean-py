Ocean class¶

class starfish.Ocean(*args, **kwargs)¶

Bases: object

The Ocean class connects to the ocean network.

For example to use this class you can do the following:

from starfish import Ocean

my_config = {
    'contracts_path': 'artifacts',
    'keeper_url': 'http://localhost:8545',
    'gas_limit': 1000,
}
ocean = Ocean(my_config)

or you can do the following setup:

from starfish.ocean import Ocean
ocean = Ocean(keeper_url='http://localhost:8545', contracts_path= artifacts, gas_limit=1000)

or with no block chain network:

from starfish.ocean import Ocean
ocean = Ocean()

You can provide these parameters in a dictionary or as individual parameters.

Parameters

  • network – name of the network to connect too. This can be ‘development’, ‘nile’ or ‘kovan’

  • keeper_url (str or None) – url to the keeper node ( http://localhost:8545 ).

  • contracts_path (str or None) – path to the contract files ( artifacts ). If not prodived then the network name will be used to search for the correct artifact files.

  • gas_limit (int or string) – The amount of gas you are willing to spend on each block chain transaction ( 0 ).

connect(keeper_url=None, network_name=None, contracts_path=None)¶

Normally you do not need to call this, since the ocean class will connect automatically using the provided keeper url and contracts path

load_account(address, password=None, keyfile=None)¶

Get an account object based on it’s address, password and keyfile. This returns the account object, of the local account.

Parameters

  • address (str, list or dict) – address of the account, if dict then use the fields, address, password and ‘keyfile’.

  • password (str) – password to access the keyfile for this account

  • keyfile (str) – JSON filename of the account key file.

Returns

return the Account object.

Type

Account or None

>>> account = ocean.load_account('0x00bd138abd70e2f00903268f3db08f2d25677c9e', 'password', 'key_file_test.json')

register_did(did, ddo, account)¶

Register this agent service with a DDO on the block chain.

Parameters

  • did (str or None) – DID to use to register for this ddo.

  • ddo – DDO to save for the registration. This is a JSON string of a DDO

  • account (Account) – account to use as the owner of the registration.

Returns

the receipt of the block chain transaction

Type

string

search_operations(text, limit=10)¶

Search the off chain storage for an operation that matches ‘text’

Parameters

  • text (str) – Search for ‘text’ in metadata.

  • limit (int) – Limit the result. If not provided, default is 10

Returns

a list of where each object is a 2-tuple (service provider did, operation did)

Type

list of 2-tuple strings

For example:

# return the first 10 records in the search for operations that do model training
#
my_result = ocean.search_operations('model_training')
