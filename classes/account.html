Account class¶

class starfish.account.Account(*args, **kwds)¶

Bases: typing.Generic

Account class, adds functionality for an account to be used on the starfish named network.

Parameters

  • address (string or dict) – address or dict of the account details

  • password (str or None) – password for the account

  • key_value (str) – key value data that is stored in the keyfile

  • keyfile (str or None) – keyfile containing the private encrypted key

If the address parameter is a string then it’s the account address. If dict then the dict can be the following format:

{
    'address': 'xxxx',
    'password': 'yyyy',
}
