Account class¶

class starfish.account.Account(ocean, address, password=None, keyfile=None, agent_adapter=None)¶

Bases: object

Account class, adds functionality for an account to be used by the Ocean network.

At the moment the Account object is created by the Ocean class.

Parameters

  • ocean (Ocean) – Ocean object

  • address (string or dict) – address or dict of the account details

  • password (str or None) – password for the account

If the address parameter is a string then it’s the account address. If dict then the dict can be the following format:

{
    'address': 'xxxx',
    'password': 'yyyy',
}
